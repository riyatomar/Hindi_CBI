बाद में बच्चन ने अपने अभिनय के कैरियर को संवारने का प्रयास किया जिसमें उसे बडे मियाँ छोटे मियाँ ( Bade Miyan Chote Miyan ) ( 1998 ) से औसत सफलता मिली और सूर्यावंशम ( Sooryavansham ) ( 1999 ) से सकारात्मक समीक्षा प्राप्त हुई लेकिन तथापि यह मान लिया गया कि बच्चन की महिमा के दिन अब समाप्त हुए चूंकि उनके बाकी सभी फिल्में जैसे लाल बादशाह ( Lal Baadshah ) ( 1999 ) और हिंदुस्तान की कसम ( Hindustan Ki Kasam ) ( 1999 ) बॉक्स ऑफिस पर विफल रही हैं ।
